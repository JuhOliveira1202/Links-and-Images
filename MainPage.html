Data Input and Output devices

The input and output of information take place trought specific devices, external or peripheral to the system. An input device can be defined as a system that allows to be entered from the outside into a computer system. On the other hand, an output device can be defined as a system that allows the computer to make information available to the outside, so that we can use it.
There are many devices that prevent the communication between the computer and the user. These devices can be grouped into three large groups:

Input devices

Output devices
